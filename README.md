# AiAdvent. Задание №4. Температура

Выполните один и тот же запрос с параметрами:

- temperature = 0;
- temperature = 0.7;
- temperature = 1.2;

Сравните ответы по:

- точности;
- креативности;
- разнообразию.

Сформулируйте:

- для каких задач лучше подходит каждая настройка.

Результат:
Примеры ответов с разной температурой и выводы по их использованию.

Формат:
Видео + Код

### Рекомендации по использованию значений тепературы

1. 0.0 Для получения ожидаемых, повторяющихся ответов. Лучше подойдет для кодирования, создания логических цепочек, строгого следования инструкциям и правилам.
2. 0.5 Баланс между двумя крайностями. Может использоваться, когда для строго варианта необходимо предоставить чуть больше свободы.
3. 1.0 Для получения разнообразных неповторяющихся ответов. Лучше подойдет для генерации гипотез, идей, брэйншторминга, стилизации текста.
